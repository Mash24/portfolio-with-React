## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mash24/jackson-portfolio.git
   cd jackson-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```

## ğŸ¨ Customization

### Personal Information
Update the following files with your information:
- `src/components/hero/Hero.jsx` - Hero section content
- `src/components/about/About.jsx` - About section
- `src/components/experience/Experience.jsx` - Work experience
- `src/components/my_skills/MySkills.jsx` - Skills and certifications
- `src/components/projects/Projects.jsx` - Portfolio projects

### Styling
- **Colors**: Update TailwindCSS color scheme in `tailwind.config.js`
- **Fonts**: Add custom fonts in `index.css`
- **Animations**: Modify Framer Motion animations in components

### SEO
- **Meta Tags**: Update `index.html` with your information
- **Structured Data**: Modify JSON-LD schema in `index.html`
- **Sitemap**: Update `public/sitemap.xml`
